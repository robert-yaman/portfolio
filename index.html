PORTFOLIO

DIG-IT

Dig-it is a music sharing app where users can “dig” songs in real time (akin to “liking”). The playback bar on the page is displayed as a two dimensional heatmap that shows which parts of the song have received the most likes. This allows users to jump to the best parts of the song. Other features include dynamic search, and a custom implementation of inifite-scroll for feeds and search results.

SNAKE

An implementation of the classic arcade game snake focussing on an elegant and simple UX. Borders are not shown until the user gets close to them. Visually appealing animations for the beginning and end of the game.

LITE RAIL

Lite Rail is a webserver MVC framework and ORM inspired by Ruby on Rails. Features include eager loading, lazy querying, CSRF protection, and flash cookies. Here, it is demoed on cat-rental app consuming complete RESTful api.
